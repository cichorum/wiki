### As an outside user, I want to easily learn what the Chicory Lane Conservation group is, what they do, what events they are holding, as well as quickly be able to access their educational resources.
- I want to be able to find exactly what I am looking for on the website quickly.
- I want information to be presented to me in an efficient and easy to follow manner.
- etc.

### As an ecologist/conservationist, I want to find and be able to access documents, resources, and information very quickly.
- I want the content I am looking for to be in a easy to reach place on the website.
- I want to be able to easily download any content I need.
- I want an AI chatbot that can provide me information about anything on the website or resources in the database.

### As the one running the website, I want the website to be structured well and allow for additions without trouble.
- I want to be able to add new pages very easily.
- I want the code base to be simple and easy to follow for even someone new taking over the website.
- I want the code utilizing the chat bot to allow for easy changes, with regards to the data it uses, the model, etc.
